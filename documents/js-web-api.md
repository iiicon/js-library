### js-web-api

| Api              | 用法                                                     | 返回值                                         |
| ---------------- | -------------------------------------------------------- | ---------------------------------------------- |
| getComputedStyle | window.getComputedStyle("元素", "伪类");                 | CSS 样式声明对象([object CSSStyleDeclaration]) |
| currentStyle     | window.currentStyle("元素", "属性")                      | css 属性（IE）                                 |
| getAttribute     | 属性名需要驼峰写法 style.getAttribute("backgroundColor") | 属性值                                         |
